Arrays

Arrays are defined like a variable. You create a variable and use "array()" as the value of the array "$variable = array(1, 2, 3)"

To fill in a variable, watch these examples:

<?php

$int = array(1, 2, 3); //int array

$string = array("s1", "s2", "s3"); //string array

You can recall arrays with a [ ] after a variable like this:

echo $string[0]; //outputs: "s1"
