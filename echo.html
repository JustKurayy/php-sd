Echo

The PHP command "echo" is like console.log in JavaScript, but instead of showing in the console, it shows it on the site. It is more compareable to the querySelector in JavaScript.

To use an "echo", watch these examples:

<?php

echo "test"; //shows "test" on the site

echo "<h1>Hello World</h1>"; //shows "Hello World" in a h1 html tag. Like the querySelector, you can add html code through echo
