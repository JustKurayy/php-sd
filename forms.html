Forms

Forms are used to get information from users. We can use the $_POST tag to pick up the information filled in the forms from input and process them.

In HTML, use the <form> tag to make a form. Inside use <input> and give it a name="" tag. With a name given to the input, the $_POST can now search for it. Make a submit button so we can send the information to the server. Instead of isset($_POST['submit']), I use $_SERVER['REQUEST_METHOD'] to see whether a request was made to the server. Because isset($_POST['submit']) can sometimes load whether or not you clicked the submit button or not.

The code below provides an example of a form with PHP:

<?php

if ($_SERVER['REQUEST_METHOD'] == "post") { //triggers if we press the submit button

echo $_POST['test']; //shows the filled input on site

};

?>
