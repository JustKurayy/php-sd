Functions

Functions in PHP are created with the "function" command. Name the function and add a "()" at the end of it.

The code below provides an example of a function that writes a message on the screen:

<?php

function writeMsg() { //created a function named "writeMsg"

echo "Success"; //shows text on site

};

writeMsg(); //shows text on site
