Superglobals

Superglobals are PHP Variables that are public to every PHP User. They are essential to basic site operating with tags like "$_POST", "$_SESSION" or "$_COOKIES"

"$_SESSION" is used to save something until the browser closes. If the user closes the tab and returns to the site, the saves session data will remain. Sessions will be removed after the user fuly closes their browser. To implement sessions into your site, add "session_start()" so that sessions can be used

"$_POST" is used to pick the data from FORMS so that the server can process it. Click on the link to learn more about "$_POST"

Cookies are made with setcookie(name, value, expiration); The "name" tag is the name of the cookie, fill value with whatever you want. Expiration is the time until the cookie is automatically removed from the user's pc. Cookies are saved, even when the browser is fully closed.

To use sessions and cookies, watch these examples:

<?php

session_start(); //opens a session in the site (IS REQUIRED)

$_SESSION['name'] = "admin"; //saves the "name" in session

setcookie("username", $_SESSION['name'], time() + 3600); //saves the cookie "name" for 1 hour (3600 seconds)

echo $_COOKIES['name']; //shows the saved cookie
